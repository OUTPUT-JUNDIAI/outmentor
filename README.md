# Conectando equipes do FIRST Tech Challenge a mentores STEAM

Missão: aproximar equipes FTC de mentores das áreas STEAM (ciência, tecnologia, engenharia, artes e matemática) para reforçar aprendizado, ética e segurança, com foco em impacto social e conformidade legal (LGPD e proteção a menores).

## 🚀 Visão geral

Para equipes: encontre mentores por especialidade, disponibilidade, idioma e modalidade (online/presencial).

Para mentores: ofereça suporte técnico (mecânica, eletrônica, programação, CAD, estratégia) e não técnico (gestão, marketing, captação, documentação).

Para organizadores: triagem, confiança e pareamento justo + métricas de impacto.

## ✨ Funcionalidades (MVP)

Cadastro de Equipe e Mentor (com verificação básica de e-mail/telefone).

Perfil com áreas STEAM, fuso, idiomas, disponibilidade semanal e modalidade.

Matching: algoritmo ponderado por compatibilidade (área + horários + idioma + modalidade + região).

Agendamento seguro via link (ex.: Google Calendar/Calendly) + registro das sessões (título, data, duração, tags).

Mensageria indireta (troca inicial via e-mail relay ou link para Slack/Discord do evento).

Painel: status das conexões, próximos encontros, feedback NPS.

Moderação e report (botão “Sinalizar”).

LGPD: consentimento, minimização de dados e política de privacidade.

Escopo explícito: a plataforma não substitui canais oficiais do FTC, nem gerencia inscrição em torneios; foca mentoria e formação.

## 👥 Perfis e papéis

Equipe (alunos + coach/mentor interno):

Define necessidades (ex.: “Programação em Java para FTC”).

Informa disponibilidade (ex.: ter/qui 19–21h).

Aprova mentores sugeridos; registra sessões; envia feedback.

Mentor (voluntário STEAM):

Define competências (ex.: CAD, ROS, Java, Eletrônica, Gestão Ágil).

Informa disponibilidade e preferências (idade/turma, online/presencial).

Aceita/recusa solicitações; envia feedback.

Admin/Organizador:

Monitora métricas, resolve conflitos, gerencia denúncias e conteúdo.

Curadoria de mentores (opcional: checagem/indicação de referências).
